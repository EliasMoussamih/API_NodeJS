# Frontend React pour API NodeJS

Ce projet React consomme l'API NodeJS présente dans le dossier `API_NodeJS`.

## Installation

1. Placez-vous dans ce dossier :
   ```bash
   cd frontend_react
   ```
2. Installez les dépendances :
   ```bash
   npm install
   ```
3. Lancez le serveur de développement :
   ```bash
   npm start
   ```

## Configuration

- L'URL de l'API est configurée dans `src/App.js` (par défaut : `http://localhost:3000/`).
- Assurez-vous que l'API NodeJS tourne sur ce port ou modifiez l'URL si besoin.
