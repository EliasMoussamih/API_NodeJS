# Backend NodeJS

Ce dossier contient le code source de l'API NodeJS.

## Lancement du backend

1. Placez-vous dans ce dossier :
   ```bash
   cd backend
   ```
2. Installez les dépendances :
   ```bash
   npm install
   ```
3. Lancez le serveur :
   ```bash
   npm start
   ```

Le backend écoute sur http://localhost:3000

---

# Structure du projet

- `backend/` : code source du backend NodeJS (API)
- `frontend_react/` : application React (frontend)
